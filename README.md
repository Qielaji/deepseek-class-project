# 智能计算器 - 在线计算工具

一个功能强大、设计精美的在线计算器Web应用，支持基础计算、科学计算和自动化公式计算。

## 🌟 功能特色

### 📱 基础计算器
- **基本运算**：支持加减乘除四则运算
- **键盘输入**：支持物理键盘输入
- **历史记录**：自动保存最近10次计算历史
- **响应式设计**：完美适配桌面和移动设备
- **错误处理**：友好的错误提示和自动恢复

### 🔬 科学计算器
- **三角函数**：sin、cos、tan及其反函数
- **对数运算**：log、ln自然对数
- **幂运算**：平方、开方、幂运算
- **数学常数**：π、e等常用常数
- **角度模式**：支持角度制和弧度制切换
- **复杂表达式**：支持括号和多层次表达式

### ⚙️ 自动化公式计算
- **欧姆定律**：电压、电流、电阻计算
- **功率计算**：电功率公式应用
- **LC谐振频率**：滤波器设计计算
- **电机转速**：交流电机转速计算
- **单位换算**：长度、重量等单位转换
- **工程公式**：常用自动化工程公式

## 🚀 快速开始

### 本地运行
1. 下载所有文件到本地目录
2. 使用Python启动HTTP服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在浏览器中访问：`http://localhost:8000`

### 在线使用
直接访问部署的在线版本：

## 💡 使用指南

### 基础计算器使用
1. **数字输入**：点击数字按钮或使用键盘输入
2. **运算符**：点击 + - × ÷ 按钮进行运算
3. **计算结果**：点击 = 按钮或按 Enter 键
4. **清除**：点击 C 按钮清除所有输入
5. **删除**：点击 DEL 按钮删除最后一个字符
6. **历史记录**：点击历史记录项可重新使用

### 科学计算器使用
1. **函数输入**：点击 sin、cos、tan 等函数按钮
2. **角度模式**：切换角度制/弧度制
3. **常数输入**：点击 π、e 等常数按钮
4. **括号使用**：支持复杂表达式计算
5. **特殊运算**：支持幂运算、开方、对数等

### 自动化公式使用
1. **选择公式**：点击所需的公式卡片
2. **输入参数**：在输入框中填入已知数值
3. **单位选择**：选择合适的单位
4. **计算结果**：点击计算按钮获得结果
5. **公式切换**：随时点击其他公式卡片切换

## ⌨️ 键盘快捷键

### 基础操作
- **数字键 (0-9)**：输入数字
- **+ - * /**：输入运算符
- **Enter 或 =**：计算结果
- **Escape 或 C**：清除所有
- **Backspace**：删除最后一个字符
- **.**：小数点

### 科学计算器
- **( )**：括号
- **^**：幂运算
